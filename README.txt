Instruções de execução:
Conversor de Bases em Python:
Para executar o programa é necessário o interpretador Python versão 3.5.1 que pode ser obtido no site oficial https://www.python.org/ .
Após a instalação , abra o arquivo "Trabalho 01 FAC.py" com o interpretador.
Também é possível abrir o programa dentro do IDLE (Editor padrão do python.)
Abra o IDLE > pressione ctrl+o ou vá em file>open e escolha o arquivo "Trabalho 01 FAC.py" , após isso pressione F5 para executar o programa.

Instruções do programa:
O programa irá pedir 3 inputs na ordem:
O número que deseja converter
A Base do número que acabou de ser escrito
A Base que deseja que o numero seja convertido para

Se o número digitado não existir na base informada o programa imprimirá a frase "Base errada" E não executará a conversão.

Se o número for digitado corretamente , será impresso o resultado na base informada no último input.
